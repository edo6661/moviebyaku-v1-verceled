# MovieByaku Application

## Technologies Used

- Formik
- Framer Motion
- React
- React Helmet
- Redux
- React Toastify
- React World Flags
- Yup
- Typescript
- RTK Query

## Features

This web application provides the following features:

- Current Movie Tracking
- Movie Reviews
- Character Information
- People Information
- Anime Information
- Ratings

## API 

- **[https://api.themoviedb.org/3/](https://api.themoviedb.org/3/)**
